## 2. 代码架构

### 2.1 架构模式

PCL CE 采用 **MVVM (Model-View-ViewModel)** 架构模式：

```
┌─────────────────────────────────────────────┐
│                   View                      │
│  (XAML: FormMain.xaml, Pages/*, Controls/*) │
└─────────────────┬───────────────────────────┘
                  │ DataBinding
┌─────────────────▼───────────────────────────┐
│               ViewModel                    │
│  (ViewModel/*, 继承 ObservableObject)        │
└─────────────────┬───────────────────────────┘
                  │ 调用
┌─────────────────▼───────────────────────────┐
│                 Model                       │
│  (Minecraft/*, IO/*, Link/*, Model/*)       │
└─────────────────────────────────────────────┘
```

## 4. 动画系统

### 4.1 动画架构

```
UI/Animation/
├── Animations.cs           # 动画定义
├── Animator.cs             # 动画器

└── WpfAnimator.cs          # WPF 动画实现
```

### 4.2 动画类型

| 动画类型 | 描述 | 示例 |
|----------|------|------|
| **FadeAnimation** | 淡入淡出 | 页面切换 |
| **ScaleAnimation** | 缩放动画 | 按钮点击效果 |
| **TranslateAnimation** | 位移动画 | 滑入效果 |
| **RotateAnimation** | 旋转动画 | 加载动画 |

### 4.3 缓动函数

支持以下缓动函数：

```csharp
public enum EasingFunction
{
    Linear,           // 线性
    EaseInQuad,       // 加速
    EaseOutQuad,      // 减速
    EaseInOutQuad,    // 先加后减
    EaseOutBack,      // 回弹效果
    EaseOutElastic    // 弹性效果
}
```

### 4.4 动画示例

```csharp
// 淡入动画
Animations.FadeIn(element, duration: TimeSpan.FromMilliseconds(300));

// 带缓动的位移动画
Animations.Translate(element, 
    fromX: 100, toX: 0, 
    easing: EasingFunction.EaseOutQuad);

// 按钮点击缩放效果
Animations.Scale(element, 
    fromScale: 1.0, toScale: 0.95, 
    duration: TimeSpan.FromMilliseconds(100),
    reverse: true); // 点击后恢复
```
